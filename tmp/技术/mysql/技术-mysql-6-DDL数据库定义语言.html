语句规范（MYSQL对大小写并不敏感，之所以要区分大小写是为了方便我们阅读区分）

  • 关键字与函数名称全部大写
  • 数据库名称、表名称、字段名称等这些自定义的全部小写
  • SQL语句必须以分号结尾

重命名表

语法：

示例：

表中添加一列

语法：（COLUMN即列，这个关键字可忽略）
ALTER TABLE tablename ADD columnname type;
或
ALTER TABLE tablename ADD COLUMN columnname type;

示例：

表中删除一列

语法：
ALTER TABLE tablesname DROP columnname;
或
ALTER TABLE tablesname DROP COLUMN columnname;

示例：

修改列的数据类型

语法：
ALTER TABLE tablename MODIFY columnname type;

示例：

语法：

或
ALTER TABLE tablename CHANGE COLUMN oldcolumnname newcolumnname type;

作用对象-服务器相关信息

功能 命令
用于显示广泛的服务器状态信息 SHOW STATUS;
显示创建特定数据库的Mysql语句 SHOW CREATE DATABASE testdatabase;
显示创建特定表的Mysql语句 SHOW CREATE TABLE testtable;
显示授予用户的安全权限（所有用户或特定用户） SHOW GRANTS;
显示服务器错误消息 SHOW ERRORS;
显示服务器警告消息 SHOW WARNINGS;
查询版本 SELECT VERSION();
查询当前日期 SELECT CURRENT_DATE;
查询当前时间 SELECT NOW();
可同时执行:
SELECT CURRENT_DATE,VERSION();
显示当前用户 SELECT USER():
修改MYSQL提示符（仅当前生效） 方法1：连接客户端时通过参数指定
mysql -u root -p --prompt 提示符
方法2：连接上客户端后，通过prompt命令修改
prompt 提示符
MYSQL常用提示符（非全部）
\D 完整的日期
\d 当前数据库
\h 服务器名称
\u 当前用户
